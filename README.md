# InteractiveHtmlBom-AD

[中文](README.zh-CN.md) | [English](README.en.md)

InteractiveHtmlBom-AD is an enhanced Interactive HTML BOM exporter for Altium
Designer, tested on AD10 and AD24.

InteractiveHtmlBom-AD 是一个面向 Altium Designer 的增强版交互式 HTML BOM
导出工具，已在 AD10 与 AD24 上验证可用。

## Quick Links

- Chinese documentation: [README.zh-CN.md](README.zh-CN.md)
- English documentation: [README.en.md](README.en.md)

## Overview

- Supports Altium Designer script workflow
- Provides GUI export and direct script export
- Includes multilingual frontend, theme improvements and usability enhancements

## 概览

- 支持 Altium Designer 脚本工作流
- 提供 GUI 导出与脚本直接导出
- 包含多语言前端、主题优化和多项易用性增强
