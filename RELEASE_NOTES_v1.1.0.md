# InteractiveHtmlBom-AD v1.1.0

## Release Notes

This release focuses on UI refinement, multilingual usability improvements, and
better README presentation for the GitHub project page.

## 发布说明

这个版本主要聚焦于界面优化、多语言易用性增强，以及 GitHub 项目首页展示效果改进。

## Highlights

- Improves the AD export window layout and interaction experience.
- Refines frontend UI details across the generated HTML viewer.
- Continues the bilingual documentation update for a clearer project homepage.
- Adds and updates screenshot assets used in the README showcase.

## 主要更新

- 优化 AD 导出窗口布局与交互体验。
- 完善导出后 HTML 查看器的前端界面细节。
- 持续完善双语文档，让项目首页展示更清晰。
- 新增并更新 README 展示截图资源。
